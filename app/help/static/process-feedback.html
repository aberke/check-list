Process a client's feedback

A client can provide you with feedback using the screen below

​

A client can select the sunny icon to indicate that you did  Great, or the cloudy icon to indicate that you did  Good, or the rainy icon to indicate that you did  OK, and finally the thundery icon to indicate that you  DID NOT DO WELL​

​

Once the client sends the feedback, you will receive a text message containing a link to the feedback.

You have to sign into your NeatStreak account to view the feedback.

You can view the feedback in the list agreement
