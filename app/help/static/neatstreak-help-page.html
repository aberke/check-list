How to use NeatStreak

Open this link on your mobile phone or type in http://neat-streak.com/ on your phone's web browser.
You will see the following screen:

 

​

 

Create a NeatStreak account

Touch the Get Started button. You will see the account creation screen

 

​

 

Type you name into the "What's your name" box

Type your phone number, eg 3479338292, into the "Your phone number?" box

Hit the Next button

You will see the screen with your name at the top 

 

​

 

Type a password into the "create a password" box

Type the same password into the "confirm password" box

Hit the Next ​button

You now have a CleanSlate account

 

View the Cleaning Dashboard

The dashboard contains the cleaning lists for all your clients. If you have just created an account,
there will be no existing lists.

 

​

 

To create a new list, hit the Create New List item in the dashboard

 

Create a New Cleaning List

When you hit Create a New List item in the dashboard screen, you will see a new screen with a section for each room
and for the client's information.

 

​

 

Hit the name of a room to open the default cleaning list for that room

 

​

 

To select which tasks should be done, touch each item you want to include or touch "select all" to include all the items...
A selected item will become highlighted.

 

​

 

To add a new item not listed, go to the text box at the bottom of the list and type a description of the new task. 

Then hit the Save button.

 

​

 

Continue on through each room that your client wants you to clean and select the relevant tasks.

Tap the "plus" sign to add another room 

 

​

 

To add notes, touch "Notes" and type in your notes in the text box then hit the Save​ button.

 

​

 

To enter your price, tap "Cost" and enter your price in the text box the hit the Save button.

 

​

Send the List to the Client

When all cleaning tasks have been selected, you can send the list to your client by tapping the "Send" button.

 

If you have not already filled in the client's contact information, you will be brought to the section where you
can fill in that information so that the new list can be sent to the client in a text message. 

 

​

 

Fill in your client's name, phone number and address and hit the Save button.

Then return to the bottom of the screen and tap the "Send" button.

The button should change to "Sent!"

 

​

 

You can return to the dashboard where you can see your new list.

 

​

Your Client can view the List

When you have sent the list, your client should receive a text message like

the following example with the agreed cleaning list.

 

​

 

When your client clicks on the link, they should see the new list along with your

note, and requested price if you provided any.

 

​

 

Your client can then also click on each room to view the agreed tasks

 

​

Delete a Cleaning List

To delete a cleaning list, go to the dashboard to view your cleaning lists. Hit the minus symbol

to the right of the list you want to delete.

 

​

 

You will see a prompt asking you if you are sure you want to delete this list

Hit OK to delete.

 

​

Sign into your account

If you already have an account, hit the Sign-In ​button

 

​

 

You will be taken to the following screen

 

​

 

Fill in your phone number and password then hit Sign-In to go to the dashboard.

 

​

Reset your password

If you forgot your password, hit Forgot password at the bottom of the screen. 

You will be taken to the following screen

 

​

 

Type in your phone number and hit the Send reset code button.

You will receive a reset code in a text message and see the following screen

 

​

 

Enter the code from the text message and your new password

Enter the same password in the Confirm password box

Hit the Reset ​button

You will return to the dashboard screen.
