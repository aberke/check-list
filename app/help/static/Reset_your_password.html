Reset your password

If you forgot your password, hit Forgot password at the bottom of the screen. 

You will be taken to the following screen

​

Type in your phone number and hit the Send reset code button.

You will receive a reset code in a text message and see the following screen

​

Enter the code from the text message and your new password

Enter the same password in the Confirm password box

Hit the Reset ​button

You will return to the dashboard screen.
